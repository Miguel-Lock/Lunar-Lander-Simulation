## Setup:
To install pygame:

```sh
pip install pygame
```

If that doesn't work because pip is not installed:
```sh
sudo apt install python3-pip
pip install pygame
```


If you encounter *error: externally-managed-environment*: 
```sh
sudo apt install python3-pygame
```
